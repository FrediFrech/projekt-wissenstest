# service/ProgressionService.java

Einfache Erklärung: Diese Klasse entscheidet, wann ein Nutzer leichter oder schwerer eingestuft wird (Platzhalter-Regeln).

## Zweck
Platzhalter‑Logik für automatische Schwierigkeits‑Progression.

## Inhalt & Verantwortung
- Schwellwerte für Promote/Demote.
- Fenstergröße für Durchschnittsberechnung.

## Verbindungen
- Später mit `config`‑Tabelle verknüpfen.
- Kann im Testflow verwendet werden, um nächste Schwierigkeit vorzuschlagen.
